Key theoretical concepts for the oral exam.

1. What is Python? ▼
Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum in 1991. It is known for its simple, English-like syntax.
2. Key Features of Python ▼
  • Simple and easy to learn
  • Interpreted (line-by-line execution)
  • Dynamically Typed (no need to declare types)
  • Open Source and Platform Independent
  • Supports OOP, Procedural, and Functional programming
3. Interpreted vs Compiled ▼
Compiled (C, Java): Converted to machine code all at once before execution.
Interpreted (Python): Executed line-by-line by an interpreter at runtime. No separate compilation step.
4. Dynamic Typing ▼
Variables do not need a type declaration. Python figures out the type (int, string, etc.) automatically when you run the code, and the type can change during execution.
5. Indentation ▼
Python uses indentation (spaces/tabs) to define code blocks instead of braces {}. Standard is 4 spaces per level. Incorrect indentation causes an IndentationError.
6. Difference between List and Tuple ▼
List: Mutable (can be changed), defined with [].
Tuple: Immutable (cannot be changed), defined with (). Tuples are faster and can be used as dictionary keys.
7. What is a Dictionary? ▼
An unordered (in older versions) collection of key-value pairs. Keys must be unique and immutable. Defined with {}.
8. Difference between / and // ▼
/ is True Division (returns float: 5/2 = 2.5).
// is Floor Division (returns integer quotient rounded down: 5//2 = 2).
9. Difference between == and is ▼
== checks value equality.
is checks identity (if they are the same object in memory).
10. What is NumPy? ▼
A library for Numerical Python. It provides fast operations on large multi-dimensional arrays and matrices.
